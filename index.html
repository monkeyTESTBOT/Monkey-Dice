HOW DO THESE VIRTUAL DICE WORK?

This website uses virtual dice created with an algorithm that assigns equal probabilities to every possible outcome. By clicking on the "ROLL AGAIN !" button you can re-roll all the dice on this page.

This page allows you to choose any number of dice between 1 and 100, each having 6 sides and 6 possible colors: Red, Orange, Yellow, Green, Blue and Purple.

ABOUT US

Monkey Dice is a fun online dice simulator that anyone can use to roll virtual color dice. With this free simulator you can roll color dice for games, decisions, or any other game of chance.
